1. Общие положения

1.1. Настоящее Пользовательское соглашение (далее — «Соглашение») регулирует отношения между владельцем сервиса «Моя Ферма» (далее — «Администрация») и пользователем (далее — «Пользователь») при использовании веб-приложения «Моя Ферма».

1.2. Используя сервис «Моя Ферма», Пользователь выражает свое полное и безоговорочное согласие со всеми условиями настоящего Соглашения.

1.3. Сервис предоставляется «как есть». Администрация не гарантирует абсолютную безошибочность и бесперебойность работы сервиса.

2. Регистрация и аккаунт

2.1. Для использования полного функционала сервиса Пользователь должен пройти процедуру регистрации, указав достоверную информацию.

2.2. Пользователь несет ответственность за сохранность своего пароля и не должен передавать его третьим лицам.

2.3. Администрация оставляет за собой право заблокировать или удалить аккаунт Пользователя в случае нарушения условий настоящего Соглашения.

3. Персональные данные

3.1. При регистрации Пользователь предоставляет следующие персональные данные: имя пользователя, адрес электронной почты, название фермы.

3.2. Администрация обязуется не передавать персональные данные Пользователя третьим лицам, за исключением случаев, предусмотренных законодательством.

3.3. Персональные данные используются исключительно для:

  • Идентификации Пользователя в системе
  • Восстановления доступа к аккаунту
  • Связи с Пользователем по вопросам работы сервиса

3.4. Пользователь имеет право в любой момент удалить свой аккаунт и все связанные с ним данные.

4. Данные о ферме

4.1. Все данные о животных, кормах, прививках и других аспектах фермерской деятельности, вносимые Пользователем, принадлежат Пользователю.

4.2. Администрация не использует Данные Пользователя в коммерческих или иных целях без явного согласия Пользователя.

4.3. Администрация принимает разумные меры для защиты Данных Пользователя от несанкционированного доступа, изменения или удаления.

5. Права и обязанности

5.1. Пользователь имеет право:

  • Использовать весь функционал сервиса в соответствии с его назначением
  • Вносить, изменять и удалять свои Данные
  • Удалить свой аккаунт в любой момент

5.2. Пользователь обязуется:

  • Не использовать сервис для нарушения законодательства
  • Не пытаться получить несанкционированный доступ к данным других пользователей
  • Не загружать вредоносный код или вирусы

6. Ограничение ответственности

6.1. Сервис «Моя Ферма» является вспомогательным инструментом для ведения учета и не заменяет профессиональные консультации ветеринаров, зоотехников и других специалистов.

6.2. Администрация не несет ответственности за убытки, связанные с использованием или невозможностью использования сервиса.

7. Изменение условий

7.1. Администрация оставляет за собой право изменять условия настоящего Соглашения.

7.2. Продолжение использования сервиса означает согласие с новой редакцией.
